# 🥷 Ninja Run

Ninja Run is a browser-based endless runner game where players control a ninja character, avoid obstacles, achieve high scores and compete on a global leaderboard.

The game combines fast-paced gameplay with score tracking and ranking functionality powered by PHP and MySQL.

---

## 🎮 Features

### 🥷 Endless Runner Gameplay

* Smooth character movement
* Obstacle avoidance mechanics
* Score progression system
* Increasing difficulty

### 🏆 Leaderboard System

* Global leaderboard
* Real-time score submission
* Player ranking system
* High-score tracking

### 📊 Score Management

* Score saving functionality
* Rank calculation
* Performance tracking
* Database-backed storage

### 🎯 Competitive Experience

* Compare scores with other players
* Track personal performance
* Climb leaderboard rankings

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/joelljoy/NINJA-RUN.git
```

### Setup Database

Import:

```text
ninja_run.sql
```

into MySQL.

### Configure Database

Update database credentials inside:

```text
db.php
```

### Run Project

Place the project inside:

```text
htdocs
```

for XAMPP.

Start:

* Apache
* MySQL

Open:

```text
http://localhost/NINJA-RUN
```
